# UniAdmin — نظام إدارة الجامعة

واجهة React (Vite) لإدارة الطلاب، المقررات، الحضور، وأعضاء هيئة التدريس — **واجهة عربية (RTL)** مع تخزين محلي للبيانات عبر Zustand persist.

## التشغيل

```bash
npm install
npm run dev
```

## الاختبارات (دوال الحساب)

```bash
npm run test
```

## ملاحظات

- **لا يوجد تصدير CSV** في الواجهة أو في طبقة `services`؛ أي تقارير لاحقة يمكن إضافتها بصيغ أخرى حسب السياسة.
- مواد الدكتور تُعرَف من **كتالوج المقررات** فقط (لا حقل نصي منفصل لقائمة المقررات عند الدكتور).
- الطلاب المسجّلون في مقرر يظهرون لدى الدكتور في **موادي** ولدى الأدمن في **ملف الطالب**.

## الهيكل

- `src/locale/ar.js` — النصوص العربية
- `src/lib/attendanceStats.js` — حساب نسب الحضور (مع اختبارات)
- `src/api/services.js` — طبقة API جاهزة للربط مع الخادم
